# Python Lab 02 - Password Strength Checker

## Overview

This beginner Python lab is a password strength checker built for IT and cybersecurity practice.

The script asks the user to enter a password, checks basic password requirements, and returns a strength rating.

## What the Script Checks

The script checks for:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers

## Strength Ratings

The password is rated as:

- Weak
- Medium
- Strong

## Skills Practiced

- Python variables
- `input()`
- `if`, `elif`, and `else`
- `for` loops
- String methods:
  - `.isupper()`
  - `.islower()`
  - `.isdigit()`
- Lists
- `.append()`
- Terminal usage
